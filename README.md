# Al-Quran-ID-Json
*Assalamu'alaikum.*
AL-QURAN Data with Translation Bahasa Indonesia, English, and Arabic Latin.

## Features
- Translation Bahasa Indonesia
- Translation English
- **Translation Arabic Latin**
